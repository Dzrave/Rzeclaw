Get Started

Up and Running in 60 Seconds

Three commands. That's all it takes.

terminal
# Install RezBot
$ npm install rezbot
 
# Initialize configuration
$ rezbot setup
✓ Config created at ~/.rezbot/config.json
✓ Memory store initialized
✓ RAG engine ready
 
# Launch the gateway
$ rezbot gateway
⚡ Gateway listening on ws://localhost:9020
🤖 Agent core online
🚀 Ready to automate!
