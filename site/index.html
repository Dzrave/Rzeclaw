Workflow

How It Works

From clone to production in four simple steps.

Clone & Build

Clone the repo, install deps, and run npm run build. That's it.

Configure

Copy the example config, plug in your API key or point to a local Ollama instance.

Launch Gateway

Start the WebSocket server. Memory, Flows, RAG, and the Agent engine spin up automatically.

Connect & Automate

Open the Web UI, Electron terminal, or use the CLI. Start automating with full AI power.
